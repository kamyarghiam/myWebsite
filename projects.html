Sleep Neural Network

This project was inspired by the sleep data that I collected for three years using an app called "Sleep Cycle." I wanted to orignally do some regular data anlaysis, but I realized I could do something more interesting: predict my sleep quality on the basis of when I sleep. My project can find the optimal amount to sleep, given the day and time, or find the best time to sleep, given how much I would to sleep and what day it is.

Deciduo

Deciduo is a program that relies on psychological inputs to learn and detect what foods/exercises make people happy but also keep them healthy. Using machine learning and Legrange optimization, the program learns what types of foods/exercises maximize people's happiness over time. You can also use other people's algorithms to see what they would prefer!

Sentiment Analysis of President Trump's Tweets

We extracted and analyzed over 37,000 tweets from president Trump. We wanted to compare the sentiments between the daytime and nighttime tweets. We used a Google API to calculate sentiment and found little to no difference between the two groups. As a note, we considered 6:00 PM - 6:00 AM to be nighttime.

Raspberry Pi/Arduino Facial Recognition

In this project, we used facial recognition to light up LED buulbs. The main modules and languages in our project were Python, OpenCV, Serial, and the Arduino programming language. The Raspberry Pi sent serial commands to an Arduino to light up the blue LED bulb when a face was recognized, the green LED bulb when eyes were recognized, and the yellow one when there was more than one face.
